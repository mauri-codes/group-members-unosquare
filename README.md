# Group Members

An application to filter members from an API endpoint 

![UI](https://user-images.githubusercontent.com/16513413/103321170-bb75f700-4a0e-11eb-8552-2f8f48343858.png)

## run locally

for the backend
```bash
cd backend
npm install
npm run start
```

for the frontend
```bash
cd frontend
npm install
npm run start
```
